Lab 2: Colors

Contrast Map

This map was created using high contrast colors selected using the Adobe Color Wheel. The distinct colors were selected by picking bright colors on opposite sides of the color wheel.

Adobe Color Wheel

Harmonious Map Version 1

This first version of a harmoneous color scheme was picked with Brown University's Color generator. The color scheme was picked to highlight trails and identify areas where trails and rivers overlap. The color green was selected for trails to reflect their connection to nature.

Brown Color Wheel

Harmonious Map Version 2

This is a second version of the harmoneous cholor scheme above. The color scheme was selected using the Media Lab's color generator. The bright purple of incorporated places was changed to a lighter purple and the thickness of trails was decreased. This is a gentler color scheme, however it is harder to differentiate the features.
