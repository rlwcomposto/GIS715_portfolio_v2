Lab 1: Symbology

ArcPro is a powerful private software that can be used to create maps. The first lab explores using ArcPy to add data and change symbologies.

View Lab

Lab 2: Colors

Colors are a powerful tool in visualization that can highlight features and trends. Color palletes were explored using several online resources and then incorporated into maps in ArcPro using ArcPy.

View Lab

Lab 3: Glyph

Glyphs display multiple pieces of information for one location. They are an instrument for communicating a lot of information for an area.

View Lab

Lab 4: Kepler

The kepler website is a free website to create time series. In this lab it was harnessed to display ecological data over a few years.

View Lab

Lab 5: Time Series

Temporal data can be displayed in a variety of ways from capturing a singular temporal stamp to displaying multiple in a gif. In this lab temporal data was analyzed in ArcPro and created into both gifs and htmls with yearly and monthly data on social media mentions of different phenomenon.

View Lab

Lab 6: Tableau

Tableau is a private software company with a free account version. This free version has many features including making interactive graphics and dashboards.
